PhotoCats

PhotoCats é uma aplicação web desenvolvida em JavaScript que consome uma API pública de imagens para gerar uma galeria dinâmica de gatos aleatórios. O projeto foi criado com foco em prática de manipulação de DOM, programação assíncrona e consumo de APIs REST utilizando Fetch API, Promises e async/await.

Funcionalidades
Gerar imagens aleatórias de gatos
Atualização dinâmica da galeria
Consumo de API externa
Manipulação de elementos HTML via JavaScript
Requisições assíncronas utilizando fetch
Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Fetch API
Promises
Async/Await
Conceitos Praticados
Manipulação do DOM
Eventos com addEventListener
Programação assíncrona
Consumo de APIs REST
Arrays e loops
Criação dinâmica de elementos HTML
Tratamento de Promises
Como Executar
Clone o repositório
Abra o projeto no VS Code
Execute utilizando Live Server
Clique no botão “Gerar Foto”
Objetivo do Projeto

O principal objetivo deste projeto foi consolidar conhecimentos fundamentais de JavaScript moderno, especialmente manipulação de DOM e operações assíncronas com APIs externas.

API Utilizada

The Cat API: https://api.thecatapi.com/v1/images/search
